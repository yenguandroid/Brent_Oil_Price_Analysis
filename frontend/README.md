# Brent Oil Dashboard — Frontend

React (Vite) + Recharts dashboard for exploring Brent oil prices, compiled market
events, and Bayesian change point results. See the main project README (two levels
up) for full setup instructions and context — this is just the frontend piece.

Quick start (requires the Flask backend running on :5000 — see `../backend/`):
```bash
npm install
npm run dev
```
Then open http://localhost:5173.
